### WeatherDemo 
<br>
实现的功能：<br>
查看城市：可以罗列出全国所有的省、市、县。<br>切换城市：滑动屏幕查看所选择的城市提拿起信息。<br>更新：背景背景图片每天会自动更新。<br>更新天气：可以手动更新天气也可以后台自动更新天气。
